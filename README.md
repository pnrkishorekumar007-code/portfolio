# Kishorekumar R - Portfolio

A modern, responsive portfolio website built with React and Vite. Showcases skills, workflow, and contact information with smooth scroll animations and a cinematic hero video.

## Features

- Cinematic hero section with background video and play/pause controls
- Animated preloader with brand reveal effect
- About and skills section with interactive ID badge design
- Services workflow with scroll-driven SVG path animation
- Contact form with parallax background typography
- Fully responsive layout for mobile, tablet, and desktop
- SEO optimized with Open Graph and Twitter meta tags
- Vercel-ready deployment configuration

## Tech Stack

- React 19, Vite 8, Tailwind CSS 4
- Framer Motion, AOS
- Deployed on Vercel

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/pnrkishorekumar007-code/new-portfolio.git
cd new-portfolio
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

### Build for Production

```bash
npm run build
npm run preview
```

### Lint

```bash
npm run lint
```

## Environment Variables

Copy `.env.example` to `.env` and configure optional variables.

## Deployment (Vercel)

1. Push this repository to GitHub
2. Import the project on Vercel
3. Vercel auto-detects Vite
4. Deploy
